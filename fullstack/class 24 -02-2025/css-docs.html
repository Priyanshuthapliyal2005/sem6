CSS Documentation with Examples

What is CSS?

CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the look and formatting of a document written in HTML or XML.

Basic Syntax

CSS syntax consists of selectors and declarations. Selectors indicate which HTML elements the styles apply to, and declarations define the styles for those elements.

            h1 {
                color: blue;
            }

Selectors

Selectors specify which HTML elements the styles apply to. They can target elements by their tag name, ID, class, or other attributes.

            /* Selects all paragraph elements */
            p {
                color: green;
            }
            
            /* Selects the element with ID 'header' */
            #header {
                font-size: 24px;
            }
            
            /* Selects all elements with class 'highlight' */
            .highlight {
                background-color: yellow;
            }

Properties and Values

Properties define what aspect of an element's style is being modified, and values specify the new style for that property.

            body {
                font-family: Arial, sans-serif;
                font-size: 16px;
            }

Box Model

The box model is a concept in CSS that describes how elements are rendered on a web page. It consists of the element's content, padding, border, and margin.

            .box {
                width: 200px;
                height: 200px;
                background-color: #f2f2f2;
                padding: 10px;
                border: 1px solid #ddd;
                margin: 20px auto;
            }

Positioning

CSS provides several ways to position elements on a web page, including static, relative, absolute, fixed, and sticky positioning.

            .relative {
                position: relative;
                top: 20px;
                left: 20px;
            }
            
            .absolute {
                position: absolute;
                top: 50%;
                left: 50%;
                transform: translate(-50%, -50%);
            }

Flexbox

Flexbox is a one-dimensional layout model that makes it easy to align and distribute space among items in a container and to make flexible adjustments to the layout.

            .container {
                display: flex;
                justify-content: center;
                align-items: center;
            }

Grid

CSS Grid is a two-dimensional layout system that allows you to create complex layouts with rows and columns.

            .grid {
                display: grid;
                grid-template-columns: repeat(3, 1fr);
                grid-template-rows: repeat(2, 1fr);
            }
